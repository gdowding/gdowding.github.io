Customizing org-page Theme

I wanted to make a more modern looking blog, since org-page. This post describes how to configure org-page with a custom theme.

I found a nice looking clean blog from Start Bootstrap. It is hosted on github, so the first step was to fork on github then clone to a local repository.

The provided thems are organized with two sub dirs

  • resources
    • css
    • fonts
    • img
    • js
  • templates

The first step was to create a directory for the new them, then copy all of the resources from startboostrap-clean-blog to the resources sub directory. The next step was to copy one of the existing templates directories.
